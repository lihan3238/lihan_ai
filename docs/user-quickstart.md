# User Quickstart

This page is the short path for invited users of Lihan AI Relay.

## What You Need

- Console: `https://api.lihan3238.com`
- API Base URL: `https://api.lihan3238.com/v1`
- A New API account that has been manually activated.
- An OpenAI-compatible client such as Chatbox, Cherry Studio, Open WebUI, Cline, or your own script.

## Five-Minute Setup

1. Log in to `https://api.lihan3238.com`.
2. Open the token/API key page.
3. Create a new API Key.
4. In your client, choose an OpenAI-compatible provider.
5. Set the base URL to `https://api.lihan3238.com/v1`.
6. Paste your API key.
7. Pick an available model and send a small test message.

## Rules

- Do not share your account or API key.
- Do not post your key in group chats.
- Do not resell access.
- Ask before running batch jobs, high concurrency, or production workloads.
- Quota shown by this site is station quota, not official USD balance.
